The `rafnixg` library is a Python package that lets you render a styled developer profile card, fetch the latest blog posts from Hashnode, and retrieve structured resume data — all from the terminal or your own Python scripts. Built on top of the [Rich](https://github.com/Textualize/rich) library, it produces clean, formatted console output with no configuration required.

## What it does

`rafnixg` bundles three areas of functionality into a single installable package. You can run it as a standalone CLI command or import individual classes into your own code. The library is designed for Python developers who want to present their profile, writing, and work history in a programmatic, terminal-friendly way.

## Key features

- **Zero configuration CLI** — run `rafnixg` after installing and get a fully formatted profile card immediately.
- **Rich console styling** — tables, colors, and highlights are handled by the [Rich](https://github.com/Textualize/rich) library.
- **Hashnode integration** — blog posts are fetched live via the Hashnode GraphQL API using the [requests](https://docs.python-requests.org/) library.
- **JSON Resume support** — the `Resume` class retrieves resume data from a hosted JSON file covering basics, work, education, skills, languages, and references.
- **Clean Python API** — all three modules (`RafnixG`, `BlogPosts`, `Resume`) are importable and usable independently in your own scripts.
- **MIT licensed** — free to use, fork, and modify.

## The three main modules

### RafnixG — personal card

The `RafnixG` class holds your profile attributes (name, position, website, GitHub, Twitter, and a short bio) and renders them as a Rich table. It also delegates to `BlogPosts` and `Links` to display posts and links in the same session.

### BlogPosts — Hashnode integration

The `BlogPosts` class queries the Hashnode GraphQL API and returns a list of `BlogPost` objects, each with a `title`, `link`, and publication metadata. You can iterate over posts or display them directly in the terminal.

### Resume — JSON Resume

The `Resume` class fetches a hosted JSON Resume file and deserializes it into typed Python objects: `Basics`, `Work`, `Education`, `Skill`, `Language`, `Certificate`, `Award`, `Project`, and `Reference`. This gives you structured access to every section of a standard resume.

## Requirements

- Python 3.9 or higher
- Dependencies installed automatically: `rich`, `requests`
