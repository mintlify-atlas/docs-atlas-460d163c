The `rafnixg` library lets you display a styled developer personal card, fetch the latest blog posts from Hashnode, and retrieve structured resume data — all from a single Python package. Whether you're using it as a CLI tool or integrating it into your own scripts, it provides a clean API built on top of the [Rich](https://github.com/Textualize/rich) library.

## Get started in 3 steps

<Steps>
  <Step title="Install the package">
    ```bash
    pip install rafnixg
    ```
  </Step>
  <Step title="Run the CLI">
    ```bash
    rafnixg
    ```
    This displays your personal card, links, and latest blog posts directly in the terminal.
  </Step>
  <Step title="Use the Python API">
    ```python
    from rafnixg import RafnixG, BlogPosts, Resume

    me = RafnixG()
    me.display()
    ```
  </Step>
</Steps>
